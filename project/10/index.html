Sicilia

What
Sicilia is a user friendly online ordering system for local chain of restaurants.
We've developed bespoke application for web and mobile devices for this project. Google maps service has been used to get geolocation and calculate distances. On this project, we got familiar with order logic, placement, cancellation and customization.
